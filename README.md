# Análisis de Ventas Walmart: Resumen Ejecutivo

## Descripción del Proyecto
Este proyecto presenta un análisis integral de los datos de ventas de Walmart, desarrollado con el objetivo de transformar datos brutos en información estratégica para la toma de decisiones. El enfoque principal fue la aplicación de técnicas de análisis financiero y operativo bajo los estándares de comunicación ejecutiva de **CFI**.

## Objetivos y Competencias Desarrolladas
* **Limpieza y Transformación de Datos:** Procesamiento de bases de datos para asegurar la integridad y precisión de la información.
* **Construcción de Dashboards Interactivos:** Diseño de tableros dinámicos en Google Sheets que permiten filtrar datos por categorías, regiones y periodos de tiempo.
* **Definición de KPIs:** Implementación de indicadores clave de desempeño (ventas totales, margen de utilidad, crecimiento mensual) para evaluar la salud del negocio.
* **Comunicación Ejecutiva:** Estructuración de reportes visuales orientados a presentaciones de alto nivel.

## Herramientas Utilizadas
* **Google Sheets:** Uso de tablas dinámicas, funciones lógicas avanzadas (VLOOKUP, QUERY, IF) y validación de datos.
* **Visualización de Datos:** Gráficos combinados y segmentadores interactivos.
